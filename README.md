## Getting Started

### Installing

Clone the repo and then install the dependencies.

1. Clone the repo:

```bash
git clone git@github.com:Noroff-FEU-Assignments/cross-course-project-norbadboy.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
